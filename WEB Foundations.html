Supplier Panel

Supplier Dashboard

B2B dashboard for sellers managing listings, orders, payments, and logistics. Brand: Crystal Indigo.

01
Foundation

Color

8 color roles on a 9-step scale. Two-tier architecture: primitives hold raw hex values; semantics hold role-based aliases organised by property.

Brand

Primary actions and elements communicating the Supplier panel identity

Success

Favourable outcomes, positive feedback, success messages

Danger

Danger, serious error states, destructive actions

Warning

Caution — preventing mistakes or errors before they occur

Neutrals

Default text, secondary UI, navigation, dividers

Progress

Progress indicators, gradient states, in-flight steps

Discovery

New features, onboarding, discovery moments

Crystal Indigo100–900

Key design decision

Primitives encode raw values (e.g. Crystal Indigo/600 = #4c31b9). Semantics encode role-based aliases by property — bg / text / border / icon / focus / overlay. Disabled states always map to Neutral/200 (#d6d6db). On-base text always maps to White.

Elevation

Three elevation levels — Low, Mid, and High Raised — each using a pair of layered shadows. All use the same colour (#1C1C22 at 16%) with varying blur and spread to create depth.
